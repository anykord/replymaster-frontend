# Replymaster Front — Full Patch

Что внутри:
- `middleware.js` — защита всех `/dashboard/**` через next-auth middleware.
- `pages/_app.js` — единая обёртка приложения: SessionProvider + авто-лейаут для всех страниц дашборда.
- `components/layouts/DashboardLayout.jsx` — шапка и левый сайдбар, общий вид как на твоём макете.
- `styles/globals.css` — базовые стили лейаута (сайдбар/шапка).
- `public/landing.html` — твой полный HTML-лендинг (не трогаем разметку).
- `pages/index.js` — страница, которая встраивает лендинг через iframe; при авторизации показывает кнопку «Перейти в Dashboard».

## Как накатить
1. Скопируй содержимое архива в корень фронтенда (согласись на замену одноимённых файлов).
2. Закоммить и задеплой.
3. Ничего в страницах дашборда править не нужно — обёртка подключится автоматически благодаря `_app.js`.
4. Если у тебя уже был `middleware.js`, можно оставить любой из вариантов с `export { default } from 'next-auth/middleware'` и `matcher: ['/dashboard/:path*']`.

## Примечания
- Если на каких-то страницах дашборда нежелателен общий лейаут, внутри файла страницы можно определить
  ```js
  Page.getLayout = (page) => page; // выключить лейаут для конкретной страницы
  ```
- Стили минимальные и не ломают существующую типографику страниц.
